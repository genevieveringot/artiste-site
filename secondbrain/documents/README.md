# Second Brain Documents

Ce dossier contient tous les documents du 2nd brain d'Hirondelle et Geneviève.

## Structure

- `concepts/` - Explorations approfondies des concepts importants
- `journal/` - Entrées de journal quotidiennes 
- `projects/` - Documentation des projets en cours
- `insights/` - Réflexions et découvertes
- `references/` - Matériel de référence

## Convention de nommage

- Concepts : `concepts/YYYY-MM-DD-nom-concept.md`
- Journal : `journal/YYYY-MM-DD.md`
- Projets : `projects/nom-projet/`
- Insights : `insights/YYYY-MM-DD-titre.md`
